**Project Type:** Individual  
**Algorithms:** AES-256-CBC · DES-CBC · RSA-2048-OAEP

## How to Run

```bash
cd crypto_project
python main.py
```

You will see a menu:

```
[1] AES-256 File Encryption & Decryption
[2] AES vs DES Algorithm Comparison
[3] Hybrid Encryption (AES data + RSA key wrap)
[q] Quit
```

Press **Enter** to use the default `data/students.csv` input, or type any file path.

---

## Modes

### Mode 1 — AES-256 File Encryption
Encrypts `students.csv` with AES-256-CBC.  
Saves ciphertext + IV + key to `output/`.  
Decrypts and verifies using SHA-256.

### Mode 2 — AES vs DES Comparison *(Path 2)*
Encrypts the same file with both AES-256-CBC and DES-CBC.  
Prints a comparison table: key size, block size, ciphertext size, timing.

### Mode 3 — Hybrid Encryption *(Path 3)*
Encrypts data with AES-256; wraps the AES key with RSA-2048 OAEP.  
Decrypts by: RSA-unwrap key → AES-decrypt data.

---

## Verification

Every mode prints a `VERIFICATION : SUCCESS` line after comparing  
SHA-256 hashes of the original and decrypted files.

---

## Notes

- DES is included **for educational comparison only** — it is cryptographically broken.  
- Input data is fully fake/generated — no real personal data used.  
- All IVs are randomly generated per encryption run (never reused).
